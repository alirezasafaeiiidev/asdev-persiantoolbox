# Dual License Policy (Planned Effective From v2.0.0)

> این سند راهنمای اجرایی است و جایگزین مشاوره حقوقی نیست.

## Scope

Persian Tools یک وب‌اپ Next.js/TypeScript با قابلیت‌های PDF، تصویر، متن، تاریخ و مالی است.

## Current Licensing State

- نسخه‌های فعلی پروژه (`<= v1.1.x`) تحت `MIT` منتشر شده‌اند.
- مجوز MIT نسخه‌های قبلی قابل بازپس‌گیری نیست.

## Planned Licensing State

از نسخه `v2.0.0` به بعد، مدل لایسنس پروژه به شکل زیر اعمال می‌شود:

- `Non-Commercial License` (رایگان)
- `Commercial License` (پولی)

این مدل از نظر OSI «Open Source» محسوب نمی‌شود و به‌عنوان `Source-available` معرفی می‌شود.

## Commercial Use Definition

هر مورد زیر استفاده تجاری محسوب می‌شود و نیازمند Commercial License است:

- ارائه سرویس SaaS یا Self-host برای مشتری
- استفاده در کار قراردادی/فریلنسری
- استفاده سازمانی داخلی برای عملیات تجاری
- درآمدزایی مستقیم یا غیرمستقیم (اشتراک، تبلیغ، اسپانسر)
- ارائه white-label یا واگذاری نسخه سفارشی

## Non-Commercial Use Definition

موارد زیر در مسیر رایگان مجاز هستند:

- استفاده شخصی، آموزشی، تحقیقاتی
- PoC یا Demo غیرتجاری
- Hosting عمومی غیرتجاری بدون تبلیغ/اشتراک/خدمت پولی

## Evaluation Window (Commercial Prospects)

- ارزیابی سازمانی غیرپروداکشن برای مدت محدود مجاز است.
- ارائه به مشتری نهایی یا استفاده درآمدزا در دوره ارزیابی مجاز نیست.

## Contribution Governance

تا قبل از نهایی شدن CLA:

- Issue/Discussion باز است.
- Merge مستقیم PR خارجی متوقف است.

پس از rollout CLA، الگوی مشارکت در `CONTRIBUTING.md` به‌روزرسانی می‌شود.
