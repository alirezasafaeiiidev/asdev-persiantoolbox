# Tools QA Matrix

این فایل به صورت خودکار از `lib/tools-registry.ts` تولید می‌شود.
برای به روزرسانی: `pnpm qa:matrix:generate`

## دسته ها

- `pdf-tools`: ابزارهای PDF
- `image-tools`: ابزارهای تصویر
- `date-tools`: ابزارهای تاریخ
- `text-tools`: ابزارهای متنی
- `finance-tools`: ابزارهای مالی

## هاب ها

- `/tools`: ابزارهای مالی آنلاین | وام، حقوق و سود بانکی - جعبه ابزار فارسی
