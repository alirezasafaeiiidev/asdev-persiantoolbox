# PTB-APP-003 — واژه‌نامه

- **Local‑First:** اپلیکیشن بدون وابستگی runtime به اینترنت جهانی و سرویس‌های ثالث کار کند.
- **Hybrid Tool:** ابزار محلی که از dataset قابل آپدیت استفاده می‌کند (آپدیت از طریق DataHub).
- **Online‑Required:** ابزاری که برای عملکرد صحیح نیازمند اینترنت است (معمولاً Pro).
- **Golden Tests:** تست‌هایی با ورودی/خروجی snapshot برای جلوگیری از رگرسیون.
- **Shell Cache:** کش شدن صفحات/فایل‌های کلیدی برای تجربه آفلاین پایدار.
