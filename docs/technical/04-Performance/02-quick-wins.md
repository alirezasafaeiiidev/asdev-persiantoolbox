# PTB-PERF-002 — اقدامات اجرایی (Quick Wins)

## اقدام 1 — Lazy Load ابزارهای PDF
- `pdfjs-dist`, `pdf-lib` باید فقط داخل route های PDF import شوند.
- در صورت امکان، هر ابزار PDF به صورت dynamic import لود شود.

## اقدام 2 — کاهش استفاده از framer-motion در فرم‌ها
- motion روی inputها و لیست‌های بزرگ حذف/کم شود.
- motion فقط برای بخش‌های نمایشی باقی بماند.

## اقدام 3 — بهینه‌سازی تصاویر و OG
- OG PNG استاندارد تولید شود.
- تصاویر سایت در صورت وجود، به فرمت‌های جدید و responsive تبدیل شوند.

## معیار پذیرش
- اندازه باندل route های غیر PDF افزایش پیدا نکند.
- صفحات PDF بعد از ورود کاربر لود شوند، نه در load اولیه سایت.
