# PTB-ARC-004 — مسیر اپ اندروید (PWA → Wrapper)

## هدف
ارائه تجربه نصب‌پذیر روی Android با حداقل هزینه نگه‌داری، سازگار با شرایط ایران.

## مسیر پیشنهادی
1) **PWA کامل** (اولویت)
   - manifest کامل
   - service worker پایدار
   - آیکن‌ها و splash درست
2) انتشار در استورهای ایرانی با Wrapper
   - TWA (اگر فقط وب کافی است)
   - Capacitor (اگر بعداً به APIهای native نیاز شد)

## الزامات برای تجربه ۱۰/۱۰
- مسیرهای Offline‑Guaranteed باید بعد از نصب بدون اینترنت کار کنند.
- صفحات Online‑Required باید پیام مناسب بدهند.
- فونت‌ها و assets بهینه باشند (حجم پایین).
