# PTB-UX-004 — دسترسی‌پذیری (Accessibility) حداقل WCAG AA

## هدف
اپ باید برای کاربران با محدودیت‌های بینایی/حرکتی نیز قابل استفاده باشد.

## بایدها (MUST)
- focus styles واضح و قابل مشاهده
- پیمایش با کیبورد (Tab order) صحیح
- منوی موبایل/Modal ها باید:
  - Focus Trap داشته باشند
  - با Esc بسته شوند
  - aria attributes صحیح داشته باشند
- فرم‌ها:
  - `aria-invalid` روی فیلدهای خطادار
  - `aria-describedby` اتصال پیام خطا به فیلد
- رنگ‌ها باید کنتراست مناسب داشته باشند (AA)

## معیار پذیرش
- تمام صفحات اصلی بدون موس قابل استفاده باشند.
- ابزارهای اصلی با screen reader قابل درک باشند (labels صحیح).
