# 06 – Finance Hub Spec (V2)

## Purpose

A category hub that:

- Lists finance tools
- Explains what each tool does
- Routes users to tools (internal links)
- Provides educational overview (local-first trust messaging)
- Includes FAQ + schema

## Content Structure

- H1: ابزارهای مالی آنلاین
- Intro (privacy-first messaging)
- Tool grid:
  - loan, salary, interest
- Guide section (1200–1800 words in V2; expand later)
- FAQ (min 5)
- Internal links to each tool page

## Schema

- CollectionPage or ItemList
- FAQPage
- BreadcrumbList

## Acceptance Criteria

- Interlinks correctly
- Valid structured data
- Good Lighthouse score
